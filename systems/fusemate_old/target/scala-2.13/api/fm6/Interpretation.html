Interpretation

class Interpretation[Time, TemporalAmount, Lit <: Lit] extends Set[Lit]

An interpretation is a set of literals, those with an explicitly given truth value as per the literal signs. The subset for a given predicate name can be accessby the forPred method. This is a form of indexing and may help speed up the computation cosiderably
