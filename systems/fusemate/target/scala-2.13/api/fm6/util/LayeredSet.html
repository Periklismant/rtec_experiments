LayeredSet

class LayeredSet[T] extends Set[T]

LayeredSet[T] represents a set of T's stored in an a priori fixed number of layers. The layeres are integer indexed, starting from 0. The elements are stored according to their index given by an indexFn. Layered sets acts like other mutable sets but offer layered access to its elements.
